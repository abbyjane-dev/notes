# CSS Formatting

## Lists

``` css
	ul,
	ol {
	  margin-top: 0.25em;
	  margin-bottom: 0.25em;
	  padding-left: 1.25em;
	}
	
	li {
	  margin-top: 0.15em;
	  margin-bottom: 0.15em;
	}
	
	p {
	  margin-bottom: 0.4em;
	}
```

## Blockquotes

``` css
	blockquote {
	  color: #111827;
	  font-style: normal;
	  border-left: 3px solid #9ca3af;
	  margin: 0.5em 0;
	  padding: 0.25em 0.75em;
	  background: none;
	  opacity: 1;
	}
```
